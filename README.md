# JRE × Michael Saylor Open Letter

Static HTML project ready to deploy on Vercel.

## Deploy on Vercel

1. Upload/import this folder to GitHub.
2. Go to Vercel → Add New Project.
3. Import the GitHub repo.
4. Framework preset: **Other**.
5. Build command: leave empty.
6. Output directory: leave empty.
7. Deploy.

## Local preview

Open `index.html` directly in your browser.

The page is fully static and includes:
- the open-letter page
- dLogos logo
- Sign Letter button
- rotating X post copies
- dLogos nomination link
